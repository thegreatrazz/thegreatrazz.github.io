Technology

This website is written in languages of the web and rendered using Jekyll. It’s a simple, serviceable site which has a landing page and a few general things.

I aim to make this website support the latest and greatest of web standards which are available in the latest Chrome/Chromium and Firefox builds. Older or non-standard browsers such as my childhood favourite are either going to have issues, or flat-out not work.

This website and the different sub-projects are hosted on GitHub Pages. And since GitHub Pages only offers HTTP/S, traffic goes through CloudFlare first.

The source code is available on GitHub.
